# 记账小助手 - 完整功能文档

## 功能概览

记账小助手是一个功能完整的本地优先个人记账应用，支持多平台账单导入、智能去重、自动分类、家庭成员管理、预算控制等高级功能。

## 技术架构

### 前端 (Renderer)
- **框架**: React 18 + TypeScript
- **状态管理**: React Hooks + URL Hash State
- **图表库**: Recharts
- **UI 组件**: 自定义组件库
- **路由**: Hash-based routing with query parameters

### 后端 (Main Process)
- **框架**: Electron Main Process + TypeScript
- **数据库**: SQLite via sql.js (embedded)
- **CSV 解析**: PapaParse
- **邮件处理**: node-imap + mailparser
- **PDF 解析**: pdf-parse + tesseract.js (OCR)
- **文件处理**: Node.js fs + child_process

### 命令行工具
- **框架**: Node.js + Commander.js
- **数据库**: sql.js (same as main app)
- **文件监控**: chokidar (optional)
- **云存储**: AWS CLI integration for S3 backup

## 数据模型

### 交易表 (transactions)
包含所有核心交易字段，支持：
- 多源数据追踪 (`source`, `original_source`, `import_id`)
- 去重标记 (`is_duplicate`, `duplicate_type`, `duplicate_source`)
- 退款关联 (`is_refund`, `refund_of`)
- 成员分配 (`member_id`)
- 标签系统 (`tags`)
- 货币支持 (`currency`)

### 扩展表
- **成员表** (`members`): 家庭成员信息
- **预算表** (`budgets`): 预算设置
- **分配历史** (`assignment_history`): 智能分配学习数据
- **分配模式** (`assignment_patterns`): 机器学习模式
- **邮箱账户** (`email_accounts`): 邮箱配置
- **邮箱消息** (`email_messages`): 邮件处理记录

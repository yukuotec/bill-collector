# 10轮持续开发计划

## 开发节奏
- **周期**: 每30分钟一轮，共10轮（5小时）
- **总结**: 每轮结束自动写总结文档
- **参考**: 对标主流记账工具（随手记、挖财、MoneyWiz、YNAB）

## 对标Feature参考

### 从随手记/挖财借鉴
- [ ] 模板记账 - 常用交易一键快速录入
- [ ] 语音记账 - 语音识别自动解析金额和分类
- [ ] 拍照记账 - 票据OCR识别
- [ ] 周期记账 - 自动记录周期性支出（房租、会员费等）
- [ ] 多人共享账本 - 家庭协作记账
- [ ] 预算超支预警 - 实时提醒和预测

### 从MoneyWiz借鉴
- [ ] 账户间转账 - 支持内部账户调拨
- [ ] 投资追踪 - 股票/基金持仓和收益计算
- [ ] 多币种自动汇率 - 实时汇率转换
- [ ] 资产负债表 - 净资产追踪

### 从YNAB借鉴
- [ ] 信封预算法 - 给每笔钱分配任务
- [ ] 目标储蓄 - 设置储蓄目标并追踪进度
- [ ] 年龄 of Money - 钱花出去前的平均持有时间

## 10轮开发规划

### 第1轮 (0-30min): UX基础优化
- Agent 1: 日期选择器优化（下拉预设）
- Agent 2: 搜索统计面板（收入/支出/结余）
- Agent 3: 重复记录批量处理

### 第2轮 (30-60min): 快速记账增强
- Agent 1: 模板记账功能
- Agent 2: 商家智能分类推荐
- Agent 3: 常用交易快捷入口

### 第3轮 (60-90min): 周期记账
- Agent 1: 周期性交易模型
- Agent 2: 自动生成周期交易逻辑
- Agent 3: 周期交易管理界面

### 第4轮 (90-120min): 预算系统升级
- Agent 1: 预算超支预警
- Agent 2: 预算使用预测
- Agent 3: 子预算支持

### 第5轮 (120-150min): 多币种支持
- Agent 1: 汇率获取服务
- Agent 2: 多币种显示逻辑
- Agent 3: 币种转换计算

### 第6轮 (150-180min): 投资追踪
- Agent 1: 投资账户模型
- Agent 2: 收益计算逻辑
- Agent 3: 投资图表展示

### 第7轮 (180-210min): 目标储蓄
- Agent 1: 储蓄目标模型
- Agent 2: 进度追踪逻辑
- Agent 3: 目标管理UI

### 第8轮 (210-240min): 数据导出增强
- Agent 1: PDF报表导出
- Agent 2: 图表导出功能
- Agent 3: 自定义导出模板

### 第9轮 (240-270min): 智能提醒
- Agent 1: 预算提醒系统
- Agent 2: 记账提醒（基于历史习惯）
- Agent 3: 数据完整性提醒

### 第10轮 (270-300min): 收尾整合
- Agent 1: Bug修复和优化
- Agent 2: 文档完善
- Agent 3: 代码清理和测试

## 每轮输出物
1. 代码提交（commit）
2. 轮次总结文档（round-X-summary.md）
3. 下一轮计划调整

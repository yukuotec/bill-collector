## 功能特性

### 1. 页面入口
- 路径: `/assign`
- 导航菜单: "分配交易"

### 2. 筛选功能
- **日期范围**: 选择起始和结束日期，筛选特定时间段的交易
- **搜索**: 按商户、描述或分类搜索交易
- **实时过滤**: 输入即时过滤交易列表

### 3. 待分配列表
- 查询 `member_id = NULL` 的交易
- 显示：金额、分类、日期、商户
- **多选支持**: 通过 checkbox 选择多笔交易
- **全选**: 一键选择所有可见（已过滤）交易
- **批量操作栏**: 选择后显示批量分配按钮

### 4. 成员盒子
- 显示所有成员（带颜色标识）
- 显示该成员已分配金额和笔数（实时更新）
- 拖拽目标区域
- **点击分配**: 有选中交易时，点击成员盒直接批量分配

### 5. 分配方式
- **拖拽分配**: 拖动交易卡片 → 放入成员盒子
- **批量分配**: 选中多笔交易 → 点击"分配给 [成员]"按钮
- **点击成员盒**: 选中交易后点击成员盒直接分配
- 分配成功后从待分配列表移除，成员统计实时更新

### 6. 批量提示
- 分配后调用 `checkSimilarAssignments`
- 如果 `shouldPrompt=true`，显示确认弹窗
- 支持一键分配相似交易

### 7. 刷新
- 刷新待分配列表和成员统计

### 相关 API
- `getTransactions({ memberId: '' })` - 获取未分配交易
- `setTransactionMember(transactionId, memberId)` - 分配成员
- `checkSimilarAssignments` - 检查相似分配
- `batchAssignSimilar` - 批量分配相似
- `getMembers` - 获取成员列表
